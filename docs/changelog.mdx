## Legacy releases

<Warning>
Versions 0.1.0 through 0.8.0 are part of Kensa's legacy implementation and have been yanked from PyPI. Kensa 0.9.0 is a complete rewrite with a new API. The old implementation remains available for historical reference.
</Warning>
